Guía de Clasificación Automática

Estados de Aprendices
DISPONIBLE

Descripción: Aprendiz disponible para vinculación laboral.

Palabras clave:

disponible activo libre buscando disponibilidad practica 1
INHABILITADO_ACT

Descripción: Aprendiz inhabilitado por actualización o suspensión.

Palabras clave:

inhabilitado suspendido bloqueado no disponible inactivo 2
PROCESO_SELECCION

Descripción: Aprendiz en proceso de selección con empresa.

Palabras clave:

seleccion proceso entrevista evaluacion evaluación en proceso 3
PROCESO_ABIERTO

Descripción: Aprendiz postulado o aplicando a vacantes.

Palabras clave:

abierto postulado aplicado candidato interesado 4
SIN_CONTRATO

Descripción: Aprendiz seleccionado pero sin contrato formalizado.

Palabras clave:

contrato pendiente esperando documentacion papeles 5
Reglas de Clasificación Automática
1. Clasificación por Palabras Clave

El sistema busca palabras clave en el campo de estado y clasifica automáticamente:

  • Si contiene "dispon", "activo", "libre" → DISPONIBLE
  • Si contiene "inhab", "suspend", "block" → INHABILITADO_ACT
  • Si contiene "selec", "proce", "entre" → PROCESO_SELECCION
  • Si contiene "abier", "post", "aplic" → PROCESO_ABIERTO
  • Si contiene "contra", "pend", "esper" → SIN_CONTRATO
2. Clasificación por Contexto

Si no hay estado definido, el sistema analiza otros campos:

  • Con empresa asignada: PROCESO_SELECCION o SIN_CONTRATO
  • Con fecha de fin práctica: DISPONIBLE
  • Sin información específica: DISPONIBLE (por defecto)
3. Clasificación por Valores Numéricos

Soporta códigos numéricos:

  • 1 → DISPONIBLE
  • 2 → INHABILITADO_ACT
  • 3 → PROCESO_SELECCION
  • 4 → PROCESO_ABIERTO
  • 5 → SIN_CONTRATO
Tips para Excel
Formato Recomendado

Usa los estados exactos para mejor precisión:

  • DISPONIBLE
  • INHABILITADO_ACT
  • PROCESO_SELECCION
  • PROCESO_ABIERTO
  • SIN_CONTRATO
Evitar Ambigüedad

No dejes el campo de estado vacío si quieres controlar la clasificación manualmente.

Verificación

Revisa el reporte de carga para ver qué aprendices fueron clasificados automáticamente.
